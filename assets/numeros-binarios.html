Explicação
Eletricidade = Linguagem
do Computador
Computadores são feitos de bilhões de transistores — minúsculos interruptores eletrônicos.

Cada transistor só pode estar:
0 = sem corrente elétrica = DESLIGADO

1 = com corrente elétrica = LIGADO

Por isso usamos base 2 (binário): apenas dois estados possíveis, como ligar/desligar uma bomba d'água ou uma estufa.

Comparação de Bases
Mesma ideia,
dígitos diferentes
Decimal (base 10): usamos 10 dígitos — 0 a 9
Quando acaba, acrescentamos uma coluna: 9 → 10

Binário (base 2): usamos 2 dígitos — 0 e 1
Quando acaba, acrescentamos uma coluna: 1 → 10

Na roça: decimal = contar dedos das mãos (10); binário = contar só presente ou ausente (2).

Potência de 2
8 = 2³ + 2³
é a escolha perfeita
Com 8 bits conseguimos representar 2⁸ = 256 valores diferentes (0 a 255).

Isso é suficiente para representar todos os caracteres do teclado: letras, números, símbolos.

Além disso, 8 é potência de 2 (2³), o que facilita cálculos eletrônicos.

Historicamente, os primeiros PCs usavam processadores de 8 bits — esse padrão ficou para sempre.

Potências de 2
Por que 2¹⁰ = 1024
vira "1 Kilo"?
No mundo decimal, 1 quilograma = 1.000 gramas exatos.
Mas em informática, 1 Kilobyte = 1.024 bytes.

Por quê essa diferença estranha?
↩ Clique para revelar
🌿
O Segredo do 1024
2¹⁰ = 1024 ≈ 1000
a fronteira natural
Computadores só "entendem" potências de 2.
A potência de 2 mais próxima de 1.000 é:
2¹⁰ = 1.024

Por convenção, chamamos isso de "1 Kilo" em informática.

Cada vez que subimos uma unidade, multiplicamos por 1.024 (≈ por mil):
KB → MB → GB → TB → PB…

É como subir degraus em uma escada — cada degrau é ×1024!

Passo a Passo
Divisões por 2
→ leia o resto de baixo pra cima
Exemplo: converter 13 para binário

13 ÷ 2 = 6, resto 1
6 ÷ 2 = 3, resto 0
3 ÷ 2 = 1, resto 1
1 ÷ 2 = 0, resto 1

Lendo os restos de baixo para cima:
1 3 = 1101 em binário

Verificação: 1×2³ + 1×2² + 0×2¹ + 1×2⁰ = 8+4+0+1 = 13 ✓

Como Funciona

Cada bit tem um peso igual a uma potência de 2.

Bit da direita = 2⁰ = 1
Próximo = 2¹ = 2
Depois = 2² = 4
... e assim por diante.

O valor decimal = soma dos pesos dos bits que estão em 1.

Combinações possíveis

Com 8 bits temos:
2⁸ = 256 combinações diferentes
(valores de 0 a 255)

É isso que representa 1 byte — suficiente para todos os caracteres do teclado!

Aplicação Agrícola

Sensores de monitoramento de solo usam binário para enviar leituras.

Por exemplo, um sensor de irrigação automática envia:
00000001 = irrigar
00000000 = parar

Simples, rápido e confiável!

Regra de Ouro

Para subir uma unidade: × 1.024
Para descer: ÷ 1.024

Aproximação prática:
× 1.000 (mas o computador prefere 1.024!)

Bit vs Byte

Bit (b minúsculo) = usado para velocidade de internet (Mbps)

Byte (B maiúsculo) = usado para tamanho de arquivo (MB, GB)

⚠️ 100 Mbps = 12,5 MB/s
(porque 100 ÷ 8 = 12,5)

Curiosidade

O DNA humano armazena ~725 MB de dados.

Uma planta de milho tem ~2,3 GB de informação genética.

A natureza usa um sistema de 4 bases (A, T, G, C) — parecido com o binário, mas em base 4!

Por que potências de 2?

O transistor (componente básico do chip) é um interruptor binário.

Por isso toda a matemática do computador é naturalmente em base 2.

Potências de 2 são os números "naturais" do hardware.

As Fronteiras

2¹⁰ = 1.024 ≈ 1K (Kilo)
2²⁰ = 1.048.576 ≈ 1M (Mega)
2³⁰ = 1.073.741.824 ≈ 1G (Giga)
2⁴⁰ ≈ 1T (Tera)

Cada salto = 10 potências a mais = × 1.024

Crescimento Exponencial

Assim como pragas na lavoura se espalham exponencialmente,
a memória dos computadores cresce em potências de 2.

Em 1980, um HD tinha 10 MB.
Hoje, HDs chegam a 20 TB.
Crescimento de 2.000.000× em 44 anos!
